mirim IT show 2018 :)

Basic - Firebase
-------------------------------
- firebase 패키지가 설치되어있지않다면
~~~
npm install -g firebase-tool
~~~

- 로그인
~~~
firebase login
Allow Firebase to collect anonymous CLI usage and error reporting information? Y
~~~

- 로그아웃
~~~
firebase logout
~~~

- 로컬 서버 실행(localhost)
~~~
firebase serve
~~~

- 서버에 deploy 
~~~
firebase deploy
~~~
